# <a name="power-bi-premium-faq"></a>Power BI 프리미엄 FAQ
Power BI 프리미엄 제품에 대한 질문과 대답 목록을 찾아보세요.

* 다른 질문이 있는 경우 [Power BI 커뮤니티에 질문합니다](http://community.powerbi.com/).
* 여전히 문제가 있나요? [Power BI 지원 페이지](https://powerbi.microsoft.com/support/)를 방문하세요.
* 또는 이 브라우저 창의 오른쪽 아래 모서리에 있는 ![](media/service-premium-faq/power-bi-contact-me-icon.png) **문의처** 아이콘을 클릭하세요.

## <a name="power-bi-premium"></a>Power BI 프리미엄
**Power BI 프리미엄이란?**  
Power BI 프리미엄은 다음이 포함된 새 용량 기반 제품입니다.

* 수신자가 라이선스를 사용자마다 개별적으로 취득할 필요 없이 엔터프라이즈 전체에서 광범위하게 보고서를 게시할 수 있는 유연성.
* Power BI 서비스의 전용 용량으로 더 커진 규모 및 성능.
* Power BI Report Server로 BI 자산 온-프레미스를 유지 관리할 수 있는 기능.
* 하나의 API 표면, 기능의 일관된 집합 및 포함된 분석을 위한 최신 기능에 대한 액세스.

**Power BI 프리미엄의 기능은? 어떻게 작동합니까?**  
Power BI 프리미엄은 각 조직에 독점적으로 할당되고 Microsoft에서 완전히 관리되는 전용 하드웨어에서 지원되는 Power BI 서비스의 용량으로 구성됩니다. 조직은 전용 용량을 광범위하게 적용하거나, 사용자 수, 워크로드 요구 또는 기타 요소에 따라 할당된 작업 영역에 할당하도록 선택할 수 있습니다. 요구 사항 변화에 따라 규모를 확장 또는 축소할 수 있습니다.

**Power BI Pro와 Power BI 프리미엄은 어떻게 다릅니까? Power BI 프리미엄이 필요한 이유는 무엇입니까?**  
Power BI 프리미엄은 용량 기반 라이선스인 반면, Power BI Pro는 사용자 기반 라이선스입니다.

Power BI 프리미엄은 각 조직에 독점적으로 할당되고, 데이터 집합 크기 및 새로 고침 비율에 대한 제한을 제거하고, 전용 하드웨어로 인한 격리를 지원하는 등 Microsoft에서 완전히 관리되는 전용 하드웨어에서 지원되는 Power BI 서비스의 용량으로 구성됩니다. Power BI 프리미엄은 또한 수신자가 라이선스를 사용자마다 개별적으로 취득할 필요 없이 엔터프라이즈 전체에서 광범위하게 보고서를 게시할 수 있는 유연성을 제공합니다.

Power BI Pro는 사용자가 보고서를 게시하고, 대시보드를 공유하고, 작업 영역에서 동료와 공동 작업하며, 다음과 같은 기타 관련된 작업을 수행하기 위한 것입니다.

* 사용자 지정된 보기의 편집 및 저장
* 개인 대시보드 만들기(새 대시보드에 고정)
* Excel 또는 Power BI Desktop에서 데이터 분석
* Excel Web App 지원과 공유
* 대시보드 공유 및 Office 365 그룹과 공동 작업
* Microsoft 팀과 함께 콘텐츠 통합

**Power BI 프리미엄을 사용하기 위해서는 Power BI Pro가 필요합니까?**  
예. Power BI Pro는 보고서를 게시하고, 대시보드를 공유하고, 작업 영역에서 동료와 공동 작업을 수행하며, 기타 관련된 작업을 수행하기 위해 필요합니다.

**Power BI 프리미엄의 비용은 얼마입니까? SKU를 얼마나 사용할 수 있습니까?**  
Power BI 프리미엄은 가상 코어 수에 따라 구입할 수 있습니다. 가격은 [Power BI 가격 책정 페이지](https://powerbi.microsoft.com/pricing/)에서 확인할 수 있습니다. 노드 및 v 코어에 자세한 내용은 [Microsoft Power BI 프리미엄 백서](https://aka.ms/pbipremiumwhitepaper)를 참조하세요. 또한 이 [계산기](https://powerbi.microsoft.com/calculator/)를 사용하여 필요한 프리미엄 용량을 예측해 보세요.

**“용량”이란 무엇을 의미합니까?**  
“용량”이란 Power BI 프리미엄 용량 관리 포털을 통해 관리자에 의해 프로비전되는 명명된 용량을 가리킵니다. 용량은 하나 이상의 노드가 모인 것입니다. 자세한 내용은 [Microsoft Power BI 프리미엄 백서](https://aka.ms/pbipremiumwhitepaper)를 참조하세요.

**Power BI 프리미엄의 청구는 어떻게 이루어집니까?**  
Power BI 프리미엄은 연간 약정으로 월별 요금이 청구됩니다.

**Power BI 프리미엄은 어떻게 구입할 수 있습니까?**  
Power BI 프리미엄은 O365 관리 센터에서 Power BI Pro에 추가된 기능으로 제공됩니다. 자세한 내용은 [Power BI 프리미엄 구매 방법](service-admin-premium-purchase.md)을 참조하세요. 자세한 내용은 Microsoft 영업 담당자에게 문의하세요.

**나에게 필요한 Power BI 프리미엄 용량은 어떻게 알 수 있습니까?**  
이 비용 [계산기](https://powerbi.microsoft.com/calculator/)를 사용하여 필요한 Power BI 프리미엄 용량을 예측해 보세요.

**Power BI 프리미엄은 언제 사용할 수 있습니까? 모든 지역에서 동시에 사용할 수 있습니까?**  
Power BI 프리미엄은 개략적으로 2017년 2분기에 현재 Power BI 서비스에서 지원하는 모든 지역에 제공될 예정입니다.

**Power BI 프리미엄은 릴리스 시점에 인증 및 규정 준수를 충족합니까? 내셔널 클라우드는 어떻습니까?**  
Power BI 프리미엄은 개략적으로 2017년 2분기에 현재 Power BI 서비스에서 지원하는 모든 지역에 제공될 예정입니다. 내셔널 클라우드, 특히 독일, 중국 및 미국 정부는 2017년 3분기에서 제공될 예정입니다. Power BI의 인증 및 규정 준수 표준은 일반 공급 시기에 Power BI 프리미엄에 적용됩니다.

**Power BI 프리미엄을 Office 365 E5와 함께 사용할 수 있습니까?**  
Power BI 프리미엄은 Power BI Pro에 추가된 기능으로 제공됩니다. Power BI Pro는 Office 365 E5에 포함됩니다. E5 고객은 기존 Pro 라이선스에 추가된 기능으로 프리미엄을 구입할 수 있습니다.

**Power BI Pro 및 Power BI 프리미엄이 최신 BI에 대해 조직에서 어떻게 작동하는지에 관한 시나리오를 간략히 알려줄 수 있습니까?**  
다음 예제에 고객이 Power BI Pro와 Power BI 프리미엄의 조합을 사용하여 어떻게 BI 요구를 충족하는지에 대해 간략히 나와 있습니다.

**Power BI Report Server란 무엇입니까? Power BI를 온-프레미스에서 사용할 수 있다는 의미입니까?**  
Power BI Report Server는 Power BI 프리미엄의 일부로 제공되며 SQL Server Reporting Services 보고서와 호환됩니다. 대화형 Power BI 보고서 및 페이지가 매겨진 보고서를 조직의 방화벽 경계 내 온-프레미스로 배포 및 분포할 수 있습니다. 조직은 클라우드에서 Power BI를 선택하거나 Power BI Report Server를 통해 온-프레미스에서 보고서를 유지하고, 원하는 대로 클라우드로 이동합니다.

Power BI 서비스의 실시간 대시보드, 자연어 쿼리 등과 같은 Power BI 기능은 Power BI Report Server의 일부로 사용할 수 없습니다. 

Power BI Report Server에 대한 자세한 내용은 [Power BI Report Server 시작](report-server/get-started.md)을 참조하세요.

**Microsoft의 Power BI Report Server 기능을 SSRS 기술 미리 보기에 대한 Power BI 보고서로 사용할 수 있습니까? 이것은 SQL의 기능으로 보였었는데 지금은 Power BI의 일부인 이유는 무엇입니까?**  
Power BI Report Server는 Power BI 프리미엄의 일부로 제공되며 SQL Server Reporting Services 보고서와 호환됩니다. 대화형 Power BI 보고서 및 페이지가 매겨진 보고서를 조직의 방화벽 경계 내 온-프레미스로 배포 및 분포할 수 있습니다.

Power BI 프리미엄의 알림은 처음으로 이 기능이 시장에 어떻게 나왔는지를 공개한 순간입니다.

Power BI Report Server에 대한 자세한 내용은 [Power BI Report Server 시작](report-server/get-started.md)을 참조하세요.

**Power BI 앱이란?**  
사용자가 엔터프라이즈 규모에서 정보를 탐색하고 검색하는 방법을 개선하기 위해 콘텐츠 팩을 Power BI 앱으로 발전시키고 있습니다. 현재 Power BI 앱은 대시보드 및 보고서를 특정 사람, 그룹 또는 전체 조직에 배포할 수 있는 간소화된 방법을 제공합니다. 비즈니스 사용자는 이러한 앱을 간편하게 설치하고 손쉽게 검색하여 한 곳으로 콘텐츠를 중앙 집중화하고 자동으로 업데이트할 수 있습니다. 

앱에 대 한 자세한 내용은 [Power BI에서 앱이란?](service-install-use-apps.md)을 참조하세요.

**Power BI 프리미엄 롤아웃의 일부로 Power BI Pro, Power BI 무료, Power BI Desktop 등에 변경 사항이 있습니까?**  
현재 Power BI Pro와 무료 서비스 간 차이를 단순화하고 있습니다. 무료 서비스는 개인 사용을 위해, Power BI Pro는 공동 작업을 위해 고안되긴 했으나, 기능적 차이점이 사용자에게 혼란을 주어왔다는 점에 대한 의견을 수렴해왔습니다. 앞으로 Power BI Pro와 동일한 기능을 갖추도록 무료 서비스를 개선할 예정이나, 공유 및 공동 작업 기능은 Power BI Pro 사용자에게만 제한됩니다.

무료 Power BI 서비스의 사용자는 모든 데이터 원본, 증가된 작업 영역 저장소 용량 한도 및 더 높은 새로 고침 및 스트리밍 속도에 대한 액세스를 이용할 수 있습니다. 이러한 변경 내용은 2017년 6월 1일부터 적용되며 [Power BI에 대한 셀프 서비스 등록](service-self-service-signup-for-power-bi.md)에서 자세히 알아볼 수 있습니다. Power BI Desktop은 무료로 계속 사용할 수 있습니다.

Extended Pro 평가판 제품에 대한 자세한 내용은 [Extended Pro 평가판 활성화](service-extended-pro-trial.md)를 참조하세요.

## <a name="power-bi-free"></a>Power BI 무료
**무료 서비스에 어떤 기능이 추가되었습니까?**  
2017년 6월 1일부터 무료 서비스는 공유, 엔터프라이즈 배포 및 공동 작업 외에 Power BI Pro와 동일한 기능이 적용됩니다. 여기에는 동일한 10GB 작업 영역 한도, 데이터 집합에 대한 최대 8일의 예약된 일별 새로 고침 및 시간당 최대 1M의 스트리밍 데이터 속도가 포함됩니다. 온-프레미스 데이터 게이트웨이를 통해 사용할 수 있는 기능을 비롯하여 모든 데이터 원본에 대한 액세스도 포함시키고 있습니다.

**Power BI Pro 기능은 무료 서비스와 함께 사용할 수 없습니까?**  
피어-투-피어 대시보드 공유, 앱 작업 영역(이전에 그룹 작업 영역이라고 함), Excel 및 Power BI 앱에서 분석(프리미엄 용량 백업 아님)은 Power BI Pro로 제한된 기능입니다.

**이러한 변경 내용은 언제부터 적용됩니까?**  
무료 서비스에 대한 변경 내용은 2017년 6월 1일부터 적용됩니다.

**무료 서비스의 영향을 받는 사용자들은 이러한 변경 내용을 어떻게 알게 됩니까?**  
2017년 6월 1일에 변경 사항으로 인해 영향을 받는 무료 서비스의 활성 사용자들은 Microsoft로부터 전자 메일 통신을 받게 되며, 더불어 2017년 5월 3일에 시작되는 Power BI 서비스에 로그인할 때 알림도 받게 됩니다.

지난 1년 내에서 활성화된 무료 서비스의 기존 사용자는 Power BI Pro의 12개월 연장 평가판을 무료로 사용할 수 있습니다. 제품은 사용자가 서비스 이용 방법에 적응할 수 있는 충분한 시간을 갖도록 Power BI Pro의 전체 기능을 제공하게 됩니다. Extended Pro 평가판 제품에 대한 자세한 내용은 [Extended Pro 평가판 활성화](service-extended-pro-trial.md)를 참조하세요.

**사용자는 그 시기에 사용자의 콘텐츠(대시보드, 보고서)에 대한 액세스를 잃게 됩니까?**  
아니요. Power BI 서비스에 업로드한 모든 콘텐츠에 대한 액세스는 손실되지 않습니다. 다른 사용자와 공유하는 콘텐츠는 2017년 6월 1일에 받는 사람이 더 이상 액세스할 수 없습니다. 마찬가지로, 그 시기에 다른 사람과 공유하는 콘텐츠에 더 이상 액세스할 수 없습니다.

지난 1년 내에서 활성화된 무료 서비스의 기존 사용자는 Power BI Pro의 12개월 연장 평가판을 무료로 사용할 수 있습니다. 제품은 사용자가 서비스 이용 방법에 적응할 수 있는 충분한 시간을 갖도록 Power BI Pro의 전체 기능을 제공하게 됩니다. Extended Pro 평가판 제품에 대한 자세한 내용은 [Extended Pro 평가판 활성화](service-extended-pro-trial.md)를 참조하세요.

**사용자가 Extended Power BI Pro 평가판을 이용 중인지 여부를 어떻게 알 수 있습니까?**  
확장된 Pro 평가판 [약관](https://aka.ms/power-bi-trial)을 참조하세요.

Extended Pro 평가판 제품에 대한 자세한 내용은 [Extended Pro 평가판 활성화](service-extended-pro-trial.md)를 참조하세요.

**사용자가 Extended Power BI Pro 평가판을 사용할 자격이 없는 경우에는 어떻게 해야 합니까?**  
[여기에서](https://app.powerbi.com/signupredirect?pbi_source=web) 표준 60일 Power BI Pro 평가판에 등록합니다.

**Power BI 프리미엄이 소개되면 무료 서비스의 기능이 변경됩니까?**  
Power BI 프리미엄의 전용 용량에 액세스 권한이 있는 무료 서비스 사용자는 게시된 앱을 통해 Power BI Pro 사용자에 의해 배포된 콘텐츠를 받을 수 있습니다.

## <a name="power-bi-pro"></a>Power BI Pro
**Power BI Pro에는 무엇이 변경됩니까?**  
Power BI Pro에 변경되는 사항은 없습니다.

**Office는 어떻습니까? Power BI Pro는 여전히 Office 365 E5에 포함되어 있습니까?**  
예. Power BI Pro는 계속 Office 365 E5에 포함될 것입니다.

**Power BI 프리미엄으로 전환하려는 경우 내 Power BI Pro 라이선스를 반환하는 옵션이 있어야 합니까?**  
아니요. Power BI 프리미엄은 Power BI Pro에 추가된 기능이므로 프리미엄에는 Pro 라이선스가 필요합니다.

**Power BI Pro가 필요한 이유는 무엇입니까? Power BI 프리미엄만 구입해야 하지 않나요?**  
Power BI 프리미엄은 용량 기반 라이선스인 반면, Power BI Pro는 사용자 기반 라이선스입니다. 공동 작업, 대시보드 공유, 임시 분석 및 보고서 게시 등 다음과 같은 기능이 필요한 사용자에게 빠르고 사용하기 쉬운 셀프 서비스 분석을 제공하기 위해 고안되었습니다.

* 사용자 지정된 보기의 편집 및 저장
* 개인 대시보드 만들기(새 대시보드에 고정)
* Excel 또는 Power BI Desktop에서 데이터 분석
* Excel 웹앱 지원으로 Excel 통합 문서 공유 및 보기
* 대시보드 공유 및 Office 365 그룹과 공동 작업
* Microsoft 팀과 함께 콘텐츠 통합

## <a name="power-bi-embedded"></a>Power BI Embedded
**Power BI Embedded의 변경 내용은 무엇입니까?**  
Power BI 프리미엄의 일환으로 Power BI Embedded를 Power BI 서비스로 결합하여 하나의 API 표면, 일관된 기능 집합 및 최신 기능에 대한 액세스를 제공합니다. 더 나아가 해당 앱에서 Power BI 포함에 관심이 있는 사용자들은 Power BI Desktop을 시작하여 Power BI 프리미엄을 사용한 배포로 이동하는 것이 좋습니다.

**앱에서 분석 포함에 대한 Power BI 프리미엄의 장점은 무엇입니까?**  
Power BI 프리미엄은 전용 하드웨어를 실행하며 Power BI 서비스의 일환으로 Microsoft에서 완전히 관리되는 용량 기반 제품입니다. 응용 프로그램 개발 수명 주기를 생각할 때 Power BI 프리미엄은 포함된 시나리오에 대한 Power BI 기능의 전체 가용성, 유연하고 예측 가능한 비용 및 개발자를 위한 향상된 콘텐츠 관리와 같은 새로운 이점을 제공합니다.

**Power BI Embedded 기반 내 기존 앱을 마이그레이션해야 합니까?**  
Power BI Embedded는 다음 일정에 대해 기존 앱에서 계속 사용할 수 있습니다.

| 라이선싱 규약 | Power BI Embedded의 가용성 |
| --- | --- |
| 기업 계약 |기존 규약의 만료 시점까지 |
| 직접 및 CSP |Power BI 프리미엄의 일반 가용성 1년 |

예, 포함된 분석 솔루션을 Power BI 프리미엄으로 마이그레이션하는 것에 대해 생각해 보아야 합니다. 자세한 내용은 [Power BI Embedded 작업 영역 컬렉션 콘텐츠를 Power BI로 마이그레이션하는 방법](developer/migrate-from-powerbi-embedded.md)을 참조하세요.

**Power BI Embedded 기반 기존 앱을 어떻게 마이그레이션할 수 있습니까?**  
앱 마이그레이션에 대한 지침은 [마이그레이션 설명서](developer/migrate-from-powerbi-embedded.md)를 참조하세요.

**Microsoft는 Power BI Embedded 무료에 대한 약정을 지키지 않습니까?**  
이전 모델에서는 개발 및 테스트를 위해 제한된 수의 세션을 제공하였습니다. 새 시나리오에서 고객은 프로덕션을 준비하는 고유한 방식으로 개발 및 테스트를 시작할 수 있는 여러 옵션을 무료로 계속 사용할 수 있습니다. 고객은 2개월 Power BI Pro 평가판을 활용하여 개발 및 테스트를 시작할 수 있습니다. Power BI Desktop의 데이터 탐색 및 보고서 제작 도구는 무료로 계속 사용할 수 있습니다.

**Power BI Embedded는 여전히 Azure Service입니까? Power BI Embedded는 어디서 구매하고 프로비전하며 관리할 수 있습니까?**  
Power BI Embedded는 Power BI 서비스와 결합되므로 앞으로 그렇게 구매, 프로비전 및 관리가 이루어질 것입니다. 자세한 내용은 [Power BI에 포함](developer/embedding.md)을 참조하세요.

## <a name="power-bi-desktop"></a>Power BI Desktop
**Power BI Desktop에 어떠한 변화가 있습니까?**  
Power BI Desktop은 변경되지 않습니다.

## <a name="power-bi-report-server"></a>Power BI Report Server
**Power BI Report Server가 별도 제품으로 판매되지 않고 Power BI 프리미엄에 포함되는 이유는 무엇입니까?**  
Power BI 프리미엄은 Power BI Report Server를 사용하여 BI 자산을 온-프레미스로 유지할 수 있는 기능을 제공합니다. Power BI Report Server는 대화형 Power BI 보고서 및 페이지가 매겨진 보고서를 조직의 방화벽 경계 내에 완전히 배포 및 분포할 수 있는 온-프레미스 서버입니다. Power BI 프리미엄을 사용하면 클라우드의 조직 프로비전에서 동일한 수의 가상 코어를 용량을 분할할 필요 없이 Power BI Report Server를 통해 온-프레미스로 배포할 수 있습니다. 조직은 클라우드에서 Power BI를 선택하거나 Power BI Report Server를 통해 온-프레미스에서 보고서를 유지하고, 원하는 대로 클라우드로 이동합니다.

자세한 내용은 [Power BI Report Server 시작](report-server/get-started.md)을 참조하세요.

**고객은 Power BI Report Server에 대한 액세스를 어떻게 얻을 수 있습니까?**  
Power BI Report Server는 Power BI 프리미엄을 통해 또는 SA와 함께 SQL Server EE를 사용하는 고객의 혜택으로 사용할 수 있습니다. 자세한 내용은 Microsoft 영업 담당자에게 문의하세요.

**Power BI Report Server는 왜 SQL Server Standard 또는 Enterprise Edition 고객 혜택으로 사용할 수 없습니까?**  
Power BI Report Server는 EE와 SA 라이선스를 사용하는 SQL Server 고객에게 제공되거나 Power BI 프리미엄을 통해 구매할 수 있습니다. Power BI는 SaaS 서비스이며, Power BI Report Server는 동일한 구독 기반 배달 모델을 따릅니다.
