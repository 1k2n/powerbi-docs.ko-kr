# <a name="supported-browsers-for-power-bi"></a>Power BI에 지원되는 브라우저
Power BI는 다음 브라우저에서 실행됩니다.

* Microsoft Edge
* Internet Explorer 11
* Chrome 데스크톱 최신 버전
* Safari Mac 최신 버전
* Firefox 데스크톱 최신 버전

이러한 브라우저는 사용할 수 있는 모든 플랫폼에서 지원됩니다.
