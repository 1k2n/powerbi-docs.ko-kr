# <a name="create-a-big-number-tile-from-a-power-bi-report"></a>Power BI 보고서에서 큰 숫자 타일 만들기
Power BI 대시보드에서 총 매출, 연간 시장 점유율, 총 영업 기회와 같이 한 가지 가장 중요한 숫자만 추적하려는 경우가 있습니다. [질문 및 답변 상자에 질문을 입력](power-bi-visualization-big-number.md)하거나 Power BI 보고서에서 큰 숫자 타일을 만들 수 있습니다. 이 문서는 보고서에서 이러한 타일을 만드는 방법에 대해 설명합니다.

1. [대시보드](service-dashboards.md)를 만들고 [데이터를 가져옵니다](service-get-data.md).
   
   연습할 데이터가 필요할 경우 [소매점 분석 샘플을 다운로드](sample-retail-analysis.md)합니다. 
2. [편집용 보기](service-reading-view-and-editing-view.md)에서 보고서를 엽니다.
3. 보고서에서 일부 공백이 있는 페이지를 찾거나 [보고서에 새 페이지를 추가](power-bi-report-add-page.md)합니다.
4. 필드 목록에서 표시하려는 숫자 필드를 선택합니다.
   
   이 예제에서는 **매장** 테이블의 **개점 수** 입니다. Power BI에서 하나의 숫자가 포함된 열 차트를 만듭니다.
   
   ![](media/power-bi-visualization-big-number-report/pbi_rptnumbertilechart.png)
5. 시각화 창에서 카드 아이콘을 선택합니다.
   
   ![](media/power-bi-visualization-big-number-report/pbi_changechartcard.png)
6. 타일을 대시보드에 추가하려면 카드를 마우스로 가리키고 고정 아이콘 ![](media/power-bi-visualization-big-number-report/pbi_pintile.png)을 선택합니다. 
   
   ![](media/power-bi-visualization-big-number-report/power-bi-pin-icon.png)
7. 기존 대시보드 또는 새 대시보드에 타일을 고정합니다. 
   
   * 기존 대시보드: 드롭다운에서 대시보드의 이름을 선택합니다.
   * 새 대시보드: 새 대시보드의 이름을 입력합니다.
8. **고정**을 선택합니다.
   
   오른쪽 위에 나타나는 성공 메시지를 통해 시각화가 타일로 대시보드에 추가되었음을 알 수 있습니다.
   
   ![](media/power-bi-visualization-big-number-report/power-bi-pin-success-message.png)
9. **대시보드로 이동**을 선택합니다. 여기에서 고정된 시각화를 [편집 및 이동](service-dashboard-edit-tile.md)할 수 있습니다.
